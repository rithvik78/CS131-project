# CS131-project

## Requirements
```sh
python3 -m pip install pycryptodome opencv-python pytesseract google-cloud-vision google-cloud-translate google-cloud-texttospeech pillow awscli boto3
```

## Steps to run 

```sh 
cd code
# fill the credentials for aws and gcp
python3 main.py
```
